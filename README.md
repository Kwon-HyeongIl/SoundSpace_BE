## 프로젝트 소개

3D 공간 음악 전시회

<br><br>

## 팀원

| **이홍주(팀장)** | **허나영** | **양동균** | **권형일** |
| :-----------: | :------: | :------: | :------: |
| FE | FE | BE | BE |

<br><br>

## 개발 환경

- 프론트엔드 : Javascript, React, Three.js
- 백엔드 : Java, Spring Boot, Spring Security, Redis, JWT, MariaDB, AWS

<br><br>

## 핵심 기능

- Genius 뮤직 API를 활용하여 음악 정보 가져오기
- 개인별 3D 공간에 본인이 원하는 음악 앨범을 전시 가능

<br><br>
